# Tổng quan dự án

Giới thiệu chung

Dự án WebSiteBanSach do đội ngũ NotFound phát triển theo kiến trúc Microservices hiện đại nhằm đáp ứng khả năng mở rộng (scalability), tính sẵn sàng cao (high availability) và khả năng chịu lỗi (fault tolerance). Hệ thống bao gồm 13 thành phần độc lập (11 backend microservices, 1 API gateway và 1 Next.js frontend client).

Toàn bộ mã nguồn Java sử dụng Spring Boot 4.0.5 và Spring Cloud 2025.1.1 hoạt động trên nền Java SDK 21.

Quy tắc Kiến trúc Bắt buộc

  • Database per Service: Mỗi service sở hữu cơ sở dữ liệu MySQL riêng biệt, tuyệt đối không truy cập chéo database.
  • JWT Gateway Validation: Gateway chịu trách nhiệm xác thực token JWT, các dịch vụ nội bộ tin tưởng header X-User-Id và X-User-Role được truyền từ API Gateway.
  • Saga Orchestrator cho Checkout: Mọi luồng checkout mua hàng bắt buộc đi qua Saga Orchestrator để đảm bảo tính nhất quán dữ liệu giữa kho, khuyến mãi, giao hàng và thanh toán.

# Quy chuẩn Thiết kế

Lưu ý khi đóng góp tài liệu đặc tả

Mọi trang đặc tả chức năng mới phải tuân thủ nghiêm ngặt các quy định về mặt giao diện để duy trì tính nhất quán:

  • Chỉ sử dụng mã màu, độ bo góc và font chữ quy định sẵn trong tệp [styles.css](styles.css).
  • Trình bày theo cấu trúc chuẩn: Tiêu đề Kicker → Mô tả nghiệp vụ ngắn → Sơ đồ luồng (Mermaid.js) → Bảng tham số I/O → Ví dụ JSON Code block.
  • Mermaid.js phải cấu hình theme variables đồng bộ với tone kem ấm của tài liệu đặc tả để tránh lòe loẹt.
